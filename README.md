# 实现Sukodu类

## 概述

本项目分别用两种语言实现了可以推理出数独空单元格候选数的Sukoku类，主要功能包括对象创建、拷贝构造、赋值构造、序列化及反序列化、棋盘推断等功能。代码中还包含了测试用例和功能使用。

## 静态语言实现

使用C++语言实现，通过以下命令编译并运行

```
g++ Sudoku.cpp -o Sudoku -g

./Sudoku.exe 
```

输入棋盘内容和棋盘大小即可进行推理。

## 动态语言实现

使用Python语言实现，通过以下命令编译并运行

```
python Sudoku.py
```

输入棋盘内容和棋盘大小即可进行推理。
